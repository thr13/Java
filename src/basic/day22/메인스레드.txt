JAVA 프로그램은 메인 스레드(main thread)가 클래스의 main() 메소드를 실행하면서 시작된다

메인 스레드는 main() 메소드의 첫 코드부터 순차적으로 실행하고 main() 메소드의 마지막 코드를 실행하거나
return 문을 만나면 실행을 종료한다

메인 스레드는 추가 작업 스레드를 만들어서 실행시킬 수 있다 (멀티 스레드처럼 하나의 프로세스내에 작업을 담당하는 스레드를 여러 개 만들 수 있다)

일반적으로 싱글 스레드는 메인 스레드 종료시 프로세스가 종료된다
그러나 멀티 스레드에서는 실행 중인 스레드가 하나라도 있다면 프로세스가 종료되지 않는다
즉, 멀티 스레드에서는 메인 스레드가 작업 스레드보다 먼저 종료되더라도 작업 스레드가 실행중이면 프로세스는 종료되지 않는다
