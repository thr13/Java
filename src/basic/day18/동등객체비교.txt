객체 해시코드는 객체를 식별하는 정수를 의미하는데
Object 클래스의 hashCode() 메소드는 객체의 메모리 번지를 이용해서 해시코드를 생성한다
그리고 해싴코드는 객체마다 다른 정수값이 리턴된다

hashCode() 메소드는 equals() 메소드와 비슷하게 두 객체가 동등한지를 비교할 때 사용된다

JAVA 는 두 객체가 동등함을 비교할때, hashCode() 와 equals() 메소드를 같이 사용한다
우선 hashCode() 가 반환하는 정수값이 같은지를 확인한 다음 equals() 메소드가 true 를 반환하는지 확인해서 동등 객체임을 판단한다

HashSet 은 동등 객체를 중복 저장하지 않는 특징을 가진다 그리고 HashSet 은 hashCode() 와 equals() 메소드를 이용해 동등 객체인지 판단한다
