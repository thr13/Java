<로컬변수(지역변수) 타입 추론>
JAVA 11 버전부터 로컬 변수를 위한 타입 추론(type inference for local variables) 기능을 사용할 수 있다
로컬 변수를 선언할 때, 예약된 타입(reserved type)인 var 를 사용하면
컴파일 시에 로컬 변수에 대입되는 값을 보고 컴파일러가 추론(inference)해서 변수의 타입을 결정한다

예) 11 버전 아래의 로컬 변수 선언 - 구체적인 타입 사용
void method() {
    String name = "김자바";
    int age = 27;
}

예) 로컬 변수를 위한 타입 추론 - var 사용
void method() {
    var name = "김자바";
    var age = 27;
}

컴파일시 각각 대입된 타입인 문자열, 정수를 컴파일러가 보고 String, int 타입으로 최종 결정된다
---
<예약된 타입 var 제한>
var 는 생성자나 메소드 안에서 로컬(지역)변수를 선언할 때만 사용할 수 있다
그러나 클래스나 인터페이스의 필드를 선언할 떄는 사용할 수 없다
---
var 는 자바의 장점인 타입 안정성을 해칠 수 있으므로 사용에 주의해야한다
만약 var 을 사용할 경우 변수명을 보고 타입을 알 수 있게끔 만들어야한다
타입으로 인한 복잡성이 증가될 경우에만 지역변수타입추론(var)을 사용하는게 좋다
