일반적으로 스레드는 자신의 이름을 가지고 있다 대표적인 예로 메인 스레드 이름은 main 이라는 이름을 가진 스레드 이다
작업스레드는 자동적으로 Thread-n 이라는 이름을 가진다 만약 작업 스레드의 이름을 변경하고 싶다면
Thread 클래스의 .setName("스레드이름") 메소드를 사용하면 된다

현재 코드가 어떤 스레드를 실행하고 있는지 확인하고싶다면 Thread 클래스의 static 메소드인 .currentThread() 로 객체의 참조 (Thread 타입)을 얻고
Thread 클래스의 .getName() 메소드로 스레드의 이름을 String 타입으로 얻을 수 있다
