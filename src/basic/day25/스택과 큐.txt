후입선출(LIFO, Last In First Out) 은 나중에 넣은 객체가 먼저 빠져나가고
선입선출(FIFO, First In First Out) 은 먼저 넣은 객체가 먼저 빠져나가는 걸 의미한다

컬렉션 프레임워크는 LIFO 자료구조를 제공하는 스택 Stack 클래스와 FIFO 자료구조를 제공하는 큐 Queue 인터페이스를 제공한다 (java.util 패키지)

스택을 응용한 대표적인 예는 JVM 스택 메모리가 있고 큐를 응용한 대표적인 예는 스레드풀(ExecutorService) 작업 큐 이다
----------------------------------------------------------------------------------------------------
객체 삽입, 삭제로는
스택은 push(), pop() 메소드를 사용하고
큐는 offer(), poll() 메소드를 사용한다
----------------------------------------------------------------------------------------------------
스택은 new Stack<>() 로 생성하고 큐는 LinkedList 객체를 이용해서 생성한다 (LinkedList 는 Queue 인터페이스를 구현한 대표적인 클래스이기 떄문)
